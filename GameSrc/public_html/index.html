Control Keys

  • Move mouse to aim laser
  • Left click to fire laser

Using The Terrain Editor
  • First you must enable the editor in the respective map file
    • Find the desired map file under 'assets/levels'
    • Find the entry for 'Editor' in the 'objectList'
    • Set 'isEnabled' to true
  • Toggle terrain visibility by pressing 1
  • Use left mouse click to select terrain blocks
  • Use TFGH (next to WASD) to move terrain
  • Use IJKL (next to TFGH) to scale terrain
  • Use arrow keys to rotate terrain
  • Use shift to slow terrain operations
  • Shift + E exports level file data to console
  • Shift + X deleted the selected terrain block
  • Shift + C clones the selected terrain block
  • Use O and P keys to zoom in and out respectively
